# Savoria Restaurant

A fully responsive multi-page website for Savoria — an upscale Italian-African fusion restaurant based in Nairobi, Kenya.

## Pages
- Home — Hero, featured dishes, why choose us, testimonials
- About — Our story, chef profiles, values
- Menu — Full menu with JavaScript category filter
- Gallery — Photo gallery of dishes and ambiance
- Reservations — Booking form with full JS validation
- Blog — Food stories and news articles
- Team — Meet the team bios
- Contact — Location, hours, contact form

## Technologies
HTML5 · CSS3 · JavaScript ES6+ · Bootstrap 5 · Bootstrap Icons

## JavaScript Features
1. Menu category filter (Starters / Mains / Desserts / Drinks)
2. Reservation form with full validation and date restrictions
3. Dark / light mode toggle saved to localStorage
